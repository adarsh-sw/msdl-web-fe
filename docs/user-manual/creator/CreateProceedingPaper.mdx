# Proceeding Paper

import Thumbnail from '@site/src/components/Thumbnail';

MSDL provides you an option to 'curate' a proceeding paper, published elsewhere. You can also publish a proceeding in
MSDL.

### Proceeding Paper Details {#proceeding-details}

This section requires the fields, as mentioned below:

<Thumbnail src="/img/proceeding/proceeding-details.png" alt="proceeding details" />

| Field                  | Description                                                                                                                         | Required |
| :--------------------- | :---------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Title                  | Title of `proceeding` in the paper.                                                                                                 |    Y     |
| Sub-title              | A sub-title is often written below the title. Mention Sub-title, if it is available.                                                |    N     |
| Language               | Select from options: English, Lao, Thai. Select multiple languages, if `proceeding` is bilingual or multilingual.                   |    Y     |
| Number of Pages/ Pages | If the given `proceeding` appears as part of a publication, mention Pages (e.g., 10-20). Else, mention the total 'Number of Pages'. |    Y     |
| License                | Select from options: Creative Commons Attribution-Non-commercial 4.0, MIT, Open Access                                              |    Y     |
| Access                 | Select from options: Open Access, Registered Users                                                                                  |    Y     |

### Proceeding Details {#proceeding-publication}

A proceeding can be published in a conference or workshop proceedings.

<Thumbnail src="/img/proceeding/proceeding-publication.png" alt="proceeding-proceeding paper  publication" />

If the 'Conference/Workshop proceeding' is not available in the list, you can type the title, and click 'Add
Conference/Workshop Proceeding'.

| Field                                 | Description                                                                                                                                                             | Required |
| :------------------------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Select Conference/Workshop proceeding | Select from the list. If the specific publication is not available in the list, you can type publication title and create it using 'Add Conference/Workshop proceeding' |    N     |

### Authors {#proceeding-authors}

In this section, mention the following details:

<Thumbnail src="/img/proceeding/proceeding-authors.png" alt="proeeding Authors" />

| :----- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |

### Publisher Details {#proceeding-publisher}

In this section, mention the following details:

<Thumbnail src="/img/proceeding/proceeding-publisher.png" alt="proceeding publisher MSDL" />

| Field                | Description                                                                                                                                                                                                                                                                            | Required |
| :------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Publisher            | Select the `Publisher` from the list, if available. If not available, type the name of the Publisher add the Publisher using 'Add Publisher' form. If you are curating a `proceeding`, provide the name of the Publisher, else mention Mekong School Digital Library as the publisher. |    Y     |
| Date of Publication  | Mention `Date of Publication` in DD/MM/YYYY format.                                                                                                                                                                                                                                    |    Y     |
| Place of Publication | Mention `Place of Publication`, such as the primary city or place mentioned as the office of the Publisher. If the city is not well-known, you may mention the district or province as well. Sometimes digital-only proceeding may not mention `Place of Publication`.                 |    N     |
| ISBN                 | Mention ISBN printed in the Book.                                                                                                                                                                                                                                                      |    N     |
| Call Number          | Publisher may provide a `Call Number` in the Catalogue of the Book.                                                                                                                                                                                                                    |    N     |
