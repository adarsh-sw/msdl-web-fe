# Book Section

import Thumbnail from '@site/src/components/Thumbnail';

MSDL provides you an option to 'curate' a Book already published in print or digital media. You can also choose to
'publish' your Book in MSDL.

### Book Details {#book-section-details}

This section requires the fields, as mentioned below:

<Thumbnail src="/img/book/book-section-details.png" alt="Curate or Publish Book Section" />

| Field            | Description                                                                                                                    | Required |
| :--------------- | :----------------------------------------------------------------------------------------------------------------------------- | :------: |
| Title            | Title of `Book` as written by the author on the cover page.                                                                    |    Y     |
| Sub-title        | A sub-title is often written above or below the title. Mention Sub-title, if it is available.                                  |    N     |
| Publication Type | Select from options: Book Section. For curating or publishing `Book`, refer to [`Book`](/latest/user-manual/reader/book-main). |    N     |
| Language         | Select from options: English, Lao, Thai. Select multiple languages, if `Blog Post` is bilingual or multilingual.               |    Y     |
| Reading Time     | Auto-fills reading time in (HH:MM:SS) format, based on `Blog Post` created using the rich text editor.                         |    Y     |
| License          | Select from options: Creative Commons Attribution-Non-commercial 4.0, MIT, Open Access                                         |    Y     |
| Access           | Select from options: Open Access, Registered Users                                                                             |    Y     |

### Publication Details {#book-section-publication}

In this section, mention the following details:

<Thumbnail src="/img/book/book-section-publication.png" alt="Curate or Publish Book section" />

A `Book Section` is part of a `Book`. So, the reference Book also needs to be created in MSDL. However, if you are
curating only the `Book Section`

| Field       | Description                                                                                                     | Required |
| :---------- | :-------------------------------------------------------------------------------------------------------------- | :------: |
| Select Book | Select the Book from the list. If the Book doesn't appear, you can type and add the Book using 'Add Book' form. |    Y     |

### Series & Volume {#book-series-volume}

In this section, mention the following details:

<Thumbnail src="/img/book/book-series-volume.png" alt="Book series and volume" />

| Field         | Description                                                                                                             | Required |
| :------------ | :---------------------------------------------------------------------------------------------------------------------- | :------: |
| Series Title  | Mention `Series Title`, if available, of the Book. Usually printed in the cover page or inside cover page of the Book.  |    N     |
| Series Number | Mention `Series Number`, if available, of the Book. Usually printed in the cover page or inside cover page of the Book. |    N     |
| Volume Title  | Mention `Volume Title`, if available, of the Book.                                                                      |    N     |
| Volume Number | Mention `Volume Number`, if available, of the Book.                                                                     |    N     |
| Edition       | Mention `Edition` of the Book.                                                                                          |    N     |

### Authors {#book-authors}

In this section, mention the following details:

<Thumbnail src="/img/book/book-authors.png" alt="Book Authors" />

| Field          | Description                                                                                                                                                           | Required |
| :------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Type of Author | Select from options: Author, Contributor, Editor, Reviewing Author, Translator. Default option selected: 'Author'.                                                    |    Y     |
| Author         | Select list of 'Authors'. If the Author is not available in the list, type the name of Author you want to add and click 'Add Author'. It will open 'Add Author' form. |    Y     |

### Publisher Details {#book-publisher}

In this section, mention the following details:

<Thumbnail src="/img/book/book-publisher.png" alt="book publisher MSDL" />

| Field                | Description                                                                                                                                                                                                                                                                      | Required |
| :------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Publisher            | Select the `Publisher` from the list, if available. If not available, type the name of the Publisher add the Publisher using 'Add Publisher' form. If you are curating a `Book`, provide the name of the Publisher, else mention Mekong School Digital Library as the publisher. |    Y     |
| Date of Publication  | Mention `Date of Publication` in DD/MM/YYYY format.                                                                                                                                                                                                                              |    Y     |
| Place of Publication | Mention `Place of Publication`, such as the primary city or place mentioned as the office of the Publisher. If the city is not well-known, you may mention the district or province as well. Sometimes digital-only Books may not mention `Place of Publication`.                |    N     |
| ISBN                 | Mention ISBN printed in the Book.                                                                                                                                                                                                                                                |    N     |
| Call Number          | Publisher may provide a `Call Number` in the Catalogue of the Book.                                                                                                                                                                                                              |    N     |

### Abstract {#book-abstract}

In this section, mention the following details:

<Thumbnail src="/img/book/book-abstract.png" alt="Book abstract" />

| Field    | Description                                                                                                                                                                                                                   | Required |
| :------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Abstract | Provide a summary of the `Book Section`, usually in 3 to 5 sentences. Provide paragraph(s), if required. A summary is usually provided by the Author or Publisher of the Book Section at the beginning of the `Book Section`. |    Y     |

In this section, mention the following details:

<Thumbnail src="/img/book/book-tags.png" alt="Blog tags" />

| :---- | :----------------------------------------------------------------------------- | :------: |

### Attachment & Link {#book-attachment-link}

In this section, mention the following details:

<Thumbnail src="/img/book/book-attachment-link.png" alt="Book attachment link MSDL" />

| Field | Description                                                                                                                                                                                                           | Required |
| :---- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| Link  | MSDL aims to provide its Readers access to content. If you are curating or publishing a `Book`, upload its PDF.                                                                                                       |    Y     |
| Link  | Mention the webpage link to the `Book Section` where it was originally published. Link is optional, because Authors and Publishers may choose to curate Book they have scanned and upload in MSDL for the first time. |    N     |
