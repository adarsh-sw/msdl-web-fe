# Dissertation/Thesis

## Introduction

A thesis (plural: theses), or dissertation (abbreviated diss.), is a document submitted in support of candidature for an
academic degree or professional qualification presenting the author's research and findings.

MSDL enables the Authors to submit Dissertation/ Thesis in PDF.

:::tip Expore Dissertation/Thesis

To get a list of `Dissertation/ Thesis`, select 'Dissertation/ Thesis' in the Resource menu in MSDL header. Readers can
further filter the list of `Dissertation/ Thesis` using the filter options: `Author`, `Publisher`, `Tags`, `Language`,
`License`, and `Access`.

:::
