# Report/ Report Section

## Introduction

MSDL provides its readers various `Reports` and `Report Section`. Readers can consult these documents in PDF.

:::tip Expore Reports

To get a list of Reports/ Report Sections, select 'Reports' as Resource Type in MSDL Explore. You can then use various
filters such as Author, Publisher, Access, License, Language, and Tags to narrow down the list of these reports and
report sections.

<Thumbnail src="/img/report/explore-report.png" alt="Report in explore" />

:::

## Report Details {#report-details}

A `Report` is cited like the following:

```
Author last name, Initials. (Year). Report title: Subtitle (Report No. number). Publisher name. URL
```

You can read in Web, Android, or iOS App.
