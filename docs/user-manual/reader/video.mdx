# Video

## Introduction

MSDL enables the Authors to curate video published and uploaded in other media and video publishing platforms, such as
YouTube.

:::tip Expore Video

To get a list of Videos, select 'Video' as Resource Type in MSDL Explore. You can then use various filters such as
Author, Publisher, Access, License, Language, and Tags to narrow down the list of these proceedings and papers.

<Thumbnail src="/img/video/video-explore.png" alt="Video in explore" />

:::

## Video Details {#video-details}

A video is cited like this:

```
Author’s last name, First name. “Video Title.” Website, Day Month Year, URL.
```

You can view video in Web, Android, or iOS App.
