# Blog Post

## Introduction

A Blog or a Web Blog is an informational website to share discrete, often informal diary-style text entries.

Blog Posts are these text entries in the Blog website. MSDL curates Blog Posts published elsewhere and lets you publish
a Blog Post, as part of your Blog.

## Explore Blog Posts

To get a list of `Blog Posts`, select 'Blog Post' in the Resource menu in MSDL header. Readers can filter the list of
`Blog Posts` using the filter options: `Author`, `Publisher`, `Tags`, `Language`, `License`, and `Access`.

<Thumbnail src="/img/blog/blog-post-resources.png" alt="Selecting Blog Post" />
