# Court Proceeding

## Introduction

`Court Proceeding` refers to all such documents those were issued by a court during its course of litigation and justice
delivery.

MSDL provides an option to curate such documents (in PDF).

:::tip Expore Court Proceeding

To get a list of `Court Proceeding`, select 'Court Proceeding' in the Resource menu in MSDL header. Readers can further
filter the list of `Court Proceeding` using the filter options: `Author`, `Publisher`, `Tags`, `Language`, `License`,
and `Access`.

<Thumbnail src="/img/court-proceedings/court-proceedings-resources.png" alt="Selecting Court proceedings" />

:::
