# Interviews

## Introduction

An interview is a structured conversation where one participant asks questions, and the other provides answers. In
common parlance, the word "interview" refers to a one-on-one conversation between an interviewer and an interviewee.

Authors, Content Creators, and Publishers can curate interviews published elsewhere in MSDL in three formats - PDF,
audio, and video. This section details interviews published in PDF.

MSDL can also publish an interview to make it available to wider audience for the first time.

:::tip Expore Interviews

To get a list of `Interviews`, select 'Interviews' in the Resource menu in MSDL header. Readers can further filter the
list of `Image/ Photograph` using the filter options: `Author`, `Publisher`, `Tags`, `Language`, `License`, and
`Access`.

<Thumbnail src="/img/interview/interview-resources.png" alt="Selecting Blog Post" />

:::
