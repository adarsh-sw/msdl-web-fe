# Map

## Introduction

A map is a symbolic representation of selected characteristics of a place, usually drawn on a flat surface.

Many maps are static, fixed to paper or some other durable medium, while others are dynamic or interactive.

:::tip Expore Map

To get a list of Maps, select 'Map' as Resource Type in MSDL Explore. You can then use various filters such as Author,
Publisher, Access, License, Language, and Tags to narrow down the list of these maps.

<Thumbnail src="/img/map/explore-map.png" alt="Selecting Journal Article in explore" />

:::

## Map Details {#map-details}

A static map appeared in other works such as books is cited like this:

```
Author. Map Title [format]. Edition. Scale. Place of publication: publisher, Date.
```

A single sheet map is cited like this:

```
Author. Title [format]. Edition. Scale. Place of publication: publisher , Date.
```

You can read in Web, Android, or iOS App.
